# Faruq Sosanya | Digital Operations & Admin Portfolio

### 🚀 "Systems over Chaos"
A high-precision, responsive portfolio built to showcase operational discipline, automated workflows, and data-driven administrative support.

---

## 🛠️ Tech Stack
* **Frontend:** React.js, Tailwind CSS, Lucide React (Icons)
* **Infrastructure:** AWS CloudFront (Asset Delivery), Vercel (Hosting/CI/CD)
* **Philosophy:** Atomic Design & "Zero-Generic" Communication

---

## 📂 Key Features
* **Selected Impact:** Interactive case studies covering INEC voter registration, SLTWG government coordination, and IMAN outreach.
* **Optimized Assets:** High-resolution professional imagery served via CloudFront for global performance.
* **Contact Grid:** Integrated communication channels via WhatsApp, Email, and LinkedIn.
